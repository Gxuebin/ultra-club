## 知识点规划

- 上篇
  - 用熟悉的 React 代码编写用户界面
  - 使用 React Hooks 重构状态管理
  - 用自带路由功能搭建多页面应用
- 下篇（✍️写作中）
  - 利用 [Taro UI](https://taro-ui.jd.com/#/) 组件库，让界面专业美观
  - 用 Redux 管理业务数据流
  - 利用小程序云进行后端开发
  - 编译为多端应用，并进行部署
